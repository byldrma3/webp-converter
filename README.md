# WebP Converter

Converts `.jpg`, `.jpeg`, `.png`, `.svg` files in your `public/` directory to `.webp`.

## 📦 Install dependencies

```bash
yarn install
```

## 🏃‍♂️ Run the script

```bash
yarn run convert
```

## ⚙️ Configuration
Edit src/index.ts to change input/output folders or supported extensions.
